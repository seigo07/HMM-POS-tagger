## Getting Started

### Dependencies

* MacOS Ventura 13.0
* Python Version: 3.9.13

### Executing program

* Please run the following command
* Usege en, fr, or uk as <lang>
```
cd CS5012_P1/src
python3 p1.py <lang>
<lang> : select one of these options: en, fr, uk.
```
